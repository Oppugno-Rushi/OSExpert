# OSExpert GitHub Pages Website

This folder is ready to be deployed via GitHub Pages.

## Quick deploy (recommended)
1. In your repo, create a folder named `docs/`.
2. Copy **everything** from this zip into `docs/`.
3. Commit + push.

Then:
- GitHub repo → **Settings** → **Pages**
- **Build and deployment**
  - Source: **Deploy from a branch**
  - Branch: `main`
  - Folder: `/docs`
- Save.

After 1–2 minutes your site will appear at:
`https://<your-username>.github.io/<repo-name>/`

For your repo `Oppugno-Rushi/OSExpert`, it will be:
`https://oppugno-rushi.github.io/OSExpert/` (case-sensitive repo name)

## Editing
- `index.html`: main content (titles, links, sections)
- `assets/css/style.css`: styling
- `assets/images/`: all figures
- `assets/files/paper.pdf`: paper PDF
